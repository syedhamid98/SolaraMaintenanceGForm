How to use Solara Maintenance Checklist Script

    Download Python installer from https://www.python.org/downloads/

    Tick "Add Python 3.x to PATH"

    Click on Customize Installation and click "Next".

    Change the PATH under "Customize install location" to "C:\Python3" (without "..." ), and wait till the installation to finish.

    Click on "Disable Max Length Path", if appeare in some cases.

    Right click on "Allow Python.bat" file and Run As Administrator to allow Python in Firewall rule.

    Right click on "Requirements.bat" file and Run As Administrator to install required modules for the Maintenance script.

    Copy "chromedriver.exe" to Google Chrome file location > "C:\Program Files (x86)\Google\Chrome\Application".

    Now double click on "Solara Maintenance.py" to start the process.

Have a good day. :)
